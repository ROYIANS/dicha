# 💡 ideas — 开发过程 idea 暂存区

这里记录开发途中冒出来的想法、纠结、待决策的产品方向。

**为什么有这个文件夹**：plan.md 铁律 #1 是「6 个月内别加新 idea」。但好想法不该丢——丢进这里，等 MVP 跑完再回来翻。**记下来 ≠ 现在做。**

## 约定

- 一条 idea 一个文件，编号前缀：`0001-<slug>.md`、`0002-<slug>.md`…
- 每篇至少写清：**背景 / 当前想法 / 待决策 / 与铁律的关系**
- 决策落定后，把结论回写进 [plan.md](../plan.md)（路线图或铁律），并在本文件标记 `✅ 已决策` 或归档。
